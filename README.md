# Automatizacion del Proceso de Onboarding con Power Automate

## Proyecto de Practica Profesional - Laboratorios Andromaco (Grupo Grunenthal)

Este proyecto lo desarrolle durante mi practica profesional para automatizar y optimizar el proceso de ingreso de nuevos colaboradores (trabajadores, practicantes y personal externo) en el departamento de Recursos Humanos de la empresa.

---

### Problematica Real

El proceso de incorporacion de personal requeria la coordinacion manual de multiples departamentos para la entrega de herramientas de trabajo. Esto generaba retrasos en la asignacion de recursos y falta de visibilidad del avance general del ingreso.

---

### Solucion Tecnologica e Integracion

Diseñe e implemente un ecosistema automatizado utilizando el entorno corporativo de Microsoft 365, logrando centralizar y agilizar las solicitudes:

* Microsoft Forms: Captura digital de los datos del nuevo colaborador y validacion de documentacion.
* Microsoft Power Automate: Motor principal encargado de procesar la logica de negocio, evaluar condiciones y distribuir las tareas.
* Microsoft SharePoint: Base de datos centralizada mediante listas estructuradas para el registro historico y auditoria.
* Microsoft Planner: Creacion automatica de tableros de trabajo con tareas asignadas a responsables especificos.
* Microsoft Outlook y Teams: Notificaciones, alertas y correos de confirmacion automatizados en tiempo real.

---

### Flujo de Tareas Automatizadas

Al enviar el formulario de ingreso, el flujo activa en paralelo las siguientes solicitudes hacia las areas correspondientes:

1. Soporte e Infraestructura TI: Solicitud de notebook y creacion del correo corporativo.
2. Accesos y Seguridad: Gestion de credenciales y fotografia del trabajador.
3. Operaciones / Logistica: Asignacion de celular corporativo y solicitud de uniforme.
4. Finanzas / Compensaciones: Registro automatico de datos bancarios para el pago de remuneraciones.

---

### Aptitudes demostradas en este proyecto

* Levantar requerimientos reales junto a jefaturas (Gerencia de Compensaciones y Beneficios).
* Integrar herramientas de productividad Cloud (Microsoft 365) para resolver ineficiencias de negocio.
* Diseñar logica de programacion visual utilizando variables, condiciones y bucles en Power Automate.
* Mejorar la trazabilidad de procesos criticos en empresas de gran escala (mas de 800 empleados).
